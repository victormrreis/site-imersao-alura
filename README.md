# Loja de EPIs — Engenharia Civil 🏗️

Site de uma loja fictícia de Equipamentos de Proteção Individual (EPIs) voltada para profissionais da construção civil.

Projeto criado durante a **Imersão Front-End com IA da Alura**.

## Funcionalidades

- Página de produtos com detalhes de capacetes, botas, óculos e luvas
- Seção institucional (Sobre Nós)
- Formulário de contato para solicitação de orçamento
- Navegação por âncoras entre seções
- Layout responsivo

## Tecnologias

- HTML5 (semântico)
- CSS3

## Como visualizar

Basta abrir o arquivo `index.html` no navegador ou acessar via GitHub Pages (se habilitado).

## Estrutura

```
├── index.html    # Página principal
├── style.css     # Estilos do site
└── README.md
```

## Contexto

Projeto desenvolvido como parte da MasterClass de Imersão Front-End com IA da Alura, com foco em estruturação semântica de HTML e estilização com CSS.
